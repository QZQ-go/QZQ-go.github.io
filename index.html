Device Functions Introduction
Real-time Monitoring
Room temperature‚ÑÉ, humidity level %, fan status real-time monitoring.
ON/OFF
ON/OFF used to turn off fan and eliminate left timer delay working status. When OFF, room temperature‚ÑÉ, humidity level % and movement detection real-time monitoring are still available.
Standby Mode
Standby Mode means fan status option you can choose when fan NOT triggered by humidity sensor or microwave sensor. There are three options for standby mode: stop, low speed, high speed.
Standby Mode Schedule
You can set Standby Mode Schedule in three different options - stop, low speed, high speed at different certain time periods at your needs.
Timer delay setting
Can be adjustable from 1-60min at intervals of 1min, works together with microwave sensor and humidity sensor.
Humidity sensor control
Humidity sensor control ON/OFF setting based on your selection, humidity level setting can be adjustable from 30% to 100% at intervals of 5%. When room humidity level increases above setting, fan status changes from one of standby mode setting to high speed, only when room humidity level decreases below setting, fan continues working in high speed according to timer delay setting, afterwards change to standby mode.
Microwave sensor control
Microwave sensor control ON/OFF setting based on your selection, movement real-time monitoring. When fan detects movement in the range of ca 5m, fan status changes from one of standby mode setting to high speed, only when no movement detected, fan continues working in high speed according to timer delay setting, afterwards change to standby mode.
Notes:
1. Microwave sensor module itself has ca 30s timer delay except individual timer delay setting.
2. Due to microwave sensor high sensitivity it can penetrate wall, wood door and glass but its sensitivity reduces to some extent.
Troubleshooting Guide
PCB Indicator Not Blinking
+

Problem: If PCB indicator not blink when first with supply power

Solution: Please connect with switch to turn on/off 5 times in 5 seconds to reset, afterwards indicator will start to blink.
Inaccurate Temperature/Humidity Data
+

Problem: When connect with App, real-time T‚ÑÉ and H% data shown in error, or T‚ÑÉ and H% shows inaccurately or stuck suddenly, it is high possibility caused by condensate water.

Solution:
1) Please first try to recheck T‚ÑÉ and H% module if fixed well with main circuit board.
2) If still in error please cut off supply power, use hair dryer to dry the white cover on the module at height of ca 15cm in less than 10s, afterwards let it cool for a while and reconnect supply power to restart fan to check if return to normal.
WiFi Connection Issues
+

Problem: If no WiFi or WiFi signal weak.

Solution: Please use second phone personal hotspot to connect fan first, then control fan only with bluetooth.
